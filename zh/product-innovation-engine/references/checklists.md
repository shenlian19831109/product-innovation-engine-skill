# 检查清单与工具速查

## 执行检查清单
- [ ] **阶段一：场景细分**：识别行业、加载框架、智能扩充、用户确认。
- [ ] **阶段二：数据采集**：覆盖 3+ 渠道、采集 200+ 条数据、记录来源情绪。
- [ ] **阶段三：数据整理**：去重、去噪、分类、情绪标注、频次统计。
- [ ] **阶段四：趋势分析**：长尾词增长、需求强度评分、新场景发现、优先级排序。
- [ ] **阶段五：需求理解**：表达形式识别、三层挖掘、理论诊断、卖点提炼。
- [ ] **阶段六：深度挖掘**：5 轮追问、根本原因、核心价值、深层爽点。
- [ ] **阶段七：产品输出**：1-6 层分层、分阶段策略、技术栈建议、执行清单。
