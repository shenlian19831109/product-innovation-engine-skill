# Checklists & Tool Quick Reference

## Execution Checklist
- [ ] **Phase 1: Scenario Segmentation**: Identify industry, load framework, intelligent expansion, user confirmation.
- [ ] **Phase 2: Data Collection**: Cover 3+ channels, collect 200+ data points, record source and emotion.
- [ ] **Phase 3: Data Organization**: De-duplicate, de-noise, classify, emotion tagging, frequency statistics.
- [ ] **Phase 4: Trend Analysis**: Long-tail growth, need intensity scoring, new scenario discovery, prioritization.
- [ ] **Phase 5: Needs Understanding**: Expression recognition, three-layer discovery, theoretic diagnosis, selling point extraction.
- [ ] **Phase 6: Deep Discovery**: 5 rounds of questioning, root cause, core value, deep delight point.
- [ ] **Phase 7: Product Output**: 1-6 layer layering, phased strategy, tech stack recommendations, execution checklist.
